# Smart Bookmark Manager

A full-stack CRUD application to manage bookmarks.

## 🔥 Features

- Add a bookmark (Title + URL)
- View all bookmarks
- Update existing bookmarks
- Delete bookmarks
- Clean and modern UI
- Fully deployed on Render

---

## 🌐 Live Application

https://smart-bookmark-manager-vy3a.onrender.com

---

## 🛠️ Tech Stack

- Backend: Python, Flask
- Database: SQLite
- Frontend: HTML, CSS, JavaScript
- Deployment: Render
- Version Control: Git & GitHub

## ⚙️ Setup Instructions

To run this project locally, first clone the repository using `git clone https://github.com/NandiniGodepurge/smart-bookmark-manager.git` and navigate into the project folder using `cd smart-bookmark-manager`. Create a virtual environment using `python -m venv venv` and activate it (on Windows: `venv\Scripts\activate`, on Mac/Linux: `source venv/bin/activate`). Then install the required dependencies using `pip install -r requirements.txt`. After installation, start the application by running `python app.py` and open your browser to `http://127.0.0.1:5000` to access the Smart Bookmark Manager.
